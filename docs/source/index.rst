parsed-sphinx-vhdl
==================

**parsed-vhdl-domain** is a `Sphinx`_ extension to that facilitates documenting VHDL projects by allowing documentation to be extracted from the code structure and its embedded comment blocks.

Installation
------------

To install::

    git clone <repo>
    cd <repo>
    pip install -e .


To build the docs::

    cd docs/
    pip install -r requirements.txt
    make html

Then open ``/build/html/index.html`` in a browser.


Usage
-----

Configure your sphinx ``conf.py`` file to include this extension and specify the root path to the VHDL code:

``doc/conf.py``

.. code-block:: python

  extensions = ['parsed_vhdl_domain']
  vhdl_root = os.path.abspath('.')

Example
-------

The restructuredText source of this file contains the following directives

.. code::

    Autogenerated Entity documentation
    **********************************

    .. vhdl:parse:: GPIO.vhd

    .. vhdl:autoentity:: GPIO


    Memory Map extracted from a comment block
    *****************************************

    .. vhdl:include-docs:: GPIO
         :start-after: GPIO Module


The results are shown below.
